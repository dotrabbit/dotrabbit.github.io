LeetCode-168.Excel Sheet Column Title

2016.01.20 dotrabbit 算法

题目

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:

　　1 -> A
　　2 -> B
　　3 -> C
　　…
　　26 -> Z
　　27 -> AA
　　28 -> AB

翻译

给出一个正整数，返回它在Excel表中对应的列名。
例如：
　　1 -> A
　　2 -> B
　　3 -> C
　　…
　　26 -> Z
　　27 -> AA
　　28 -> AB

解

其实就是把数字转换为26进制数，并用大写字母表示。
