简明Python教程-1.介绍

2016.03.20 dotrabbit Python

1.1 简介

简单，功能强大。
注：Python语言的创造者Guido van Rossum是根据英国广播公司的节目“蟒蛇飞行马戏”命名这个语言的。

1.2 Python的特色

  • 简单
    编写类似伪代码，专注于解决问题，而不是语言本身
  • 易学
    语法简单，极易上手
  • 免费、开源
    FLOSS(自由/开放源码软件)之一
  • 高层语言
    无需考虑诸如如何管理内存一类的底层细节
  • 可移植性
    开源，多平台
  • 解释性
    Python写的程序不需要编译成二进制代码。可以直接从源代码运行程序。
    Python解释器把源代码转换成字节码，再把它翻译成计算机使用的语言(二进制代码，即0和1)并运行。
    无需担心如何编译程序、如何确保连接转载正确的库等等。
    只要把你的Python程序拷贝到另外一台计算机上，就可以工作了，使得你的Python程序更加易于移植。
  • 面向对象
    Python既支持面向过程的编程，也支持面向对象的编程。
  • 可扩展性
    如果希望某些关键代码运行得更快或者某些算法不公开，可以将部分程序用C或C++编写，然后在Python程序中使用它们。
  • 丰富的库
    标准库庞大

1.3 为什么不适用Perl？

Perl程序简单，但不适用大型程序。
Python更简单、清晰、易于编写、易懂、易维护。
Python的库也在慢慢扩大。
